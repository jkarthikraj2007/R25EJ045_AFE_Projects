# Project 4: Live News Website

This folder contains project 4 for the AI Foundation for Engineers assignment.

## Features

- Responsive news homepage
- Featured headline section
- Live updated timestamp
- Breaking news ticker
- Category filters
- Search box
- Dynamic news cards rendered with JavaScript

## How to Run

Open `index.html` in a browser.
